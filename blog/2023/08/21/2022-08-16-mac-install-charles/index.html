Charles For Mac 安装与使用

Charles 可以说是客户端的必备软件，经常用来做网络请求数据包的捕获。

1、下载地址如下：
链接：https://pan.baidu.com/s/1wB57wfWAJvMIY2leO1Cmww?pwd=p881
提取码：p881
2、安装与破解步骤
下载之后会得到两个文件，charles-proxy-4.2.7.dmg 和 charles.jar
（1）双击 charles-proxy-4.2.7.dmg 安装包，一步步完成安装即可，安装完成之后正常打开一次Charles，再完全退出。
（2）将第二个文件 charles.jar 复制到 /Applications/Charles.app/Contents/Java/ 目录下，如果已存在，则直接覆盖即可。
（3）重新打开Charles，即可看到已破解，并能正常使用。
3、使用
（1）环境配置
如果电脑的 ip 是 192.168.99.15，则手机连上与电脑同一网络后，设置代理ip也是192.168.99.15，并设置端口号为8888
（2）电脑打开 Charles，测试手机能否正常上网
（3）手机打开浏览器，输入 http://chls.pro/ssl，完成描述文件的安装
（4）手机设置证书信任：设置 —> 通用 —-> 关于手机，拉到最后一项，点进去，信任对应的证书
ps：如果手机不能正常上网，建议重启手机，重述上述步骤即可。
4、可能会遇到的问题
（1）抓取 https 请求时显示乱码
解决方案：点击 Charles 菜单栏 Proxy —> SSL Proxying Setting，在弹出的窗口中点击 Add，在新的窗口 Host 一栏中输入 *，点 OK 保存。重启 Charles 即可。步骤如下图所示：
