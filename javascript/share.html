# 深入理解函数参数传值

# 定义

ECMAScript中所有函数的参数都是按值传递。

# 按值传递

  • 定义

把函数外部的值复制给函数内部的参数。就像把值从一个变量复制到另一个变量。

  • 举个栗子
    var a = 1;
    function fn(a) {
        a = 2;
        console.log(a);  // 2
    }
    fn(a);
    console.log(a); // 1

当传递参数a到函数fn时，相当于拷贝了变量a， 所以修改a中的值不会影响原来的值

# 按引用传递

  • 举个栗子
    var obj = {
        a: 1
    }
    function fn(obj) {
        obj.a = 2;
        console.log("result1___", obj);
    }
    fn(obj);
    console.log("result2___", obj);

修改了obj对象a属性的值，这是为什么，说好的参数按值传递呢

# 按共享传递

传递对象的时候，传递的是对象引用的副本

  • 再举个栗子
 var obj = {
    a: 1
}
function fn(obj) {
    obj = 2;
    console.log("result1___", obj);  // { a: 2}
}
fn(obj);
console.log("result2___", obj);  // {a: 1}

注意：按引用传递是传递对象的引用，按共享传递是传递对象引用的副本
