# 数组扁平化

# 扁平化示例

数组的扁平化，就是将嵌套多层的数组array转换为只有一层的数组，示例代码如下：

var arr = [1, [2, 3, [5, 6]]]
// 扁平化后如下
console.log(flattern(arr));  // [1,2,3,5,6]
